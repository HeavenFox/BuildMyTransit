BuildMyTransit.nyc
Interactive Transit System Designer & Simulator

What is this?

BuildMyTransit.nyc is a web app to design, visualize, and simulate New York City subway systems. Perfect for exploring "what-if" scenarios.

Where did the data come from?

Data is sourced from OpenStreetMap.

I noticed an error in the data!

The best way to fix any data error is by fixing it upstream in OpenStreetMap. This way all OSM users can benefit.

Where are the custom routes stored?

In your browser, so they will still be here if you leave and return. Unless you are using incognito, or cleared your browser data.

Can I use it for my YouTube video / presentation / ...?

Generally yes, but attribution to OpenStreetMap is required. We recommend the following form:

Data © OpenStreetMap. Visualization by BuildMyTransit.nyc.

Is it open source?

Yes. BuildMyTransit.nyc is Apache 2.0 licensed and available on GitHub, except OpenStreetMap data, which is licensed under the Open Database License (ODbL).
